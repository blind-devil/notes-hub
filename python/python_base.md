### <a name='芯锌械褉邪褌芯褉褘_锌褉懈褋胁邪懈胁邪薪懈褟'></a> 袨锌械褉邪褌芯褉褘 锌褉懈褋胁邪懈胁邪薪懈褟 [馃爼](#馃爼)

* `=` - 锌褉懈褋胁邪懈胁邪薪懈械: `x = 5`.
* `+=` - 写芯斜邪胁谢械薪懈械 褋 锌褉懈褋胁邪懈胁邪薪懈械屑: `x += 3`.
* `-=` - 胁褘褔懈褌邪薪懈械 褋 锌褉懈褋胁邪懈胁邪薪懈械屑: `x -= 3`.
* `*=` - 褍屑薪芯卸械薪懈械 褋 锌褉懈褋胁邪懈胁邪薪懈械屑: `x *= 3`.
* `/=` - 写械谢械薪懈械 褋 锌褉懈褋胁邪懈胁邪薪懈械屑: `x /= 3`.
* `//=` - 褑械谢芯褔懈褋谢械薪薪芯械 写械谢械薪懈械 褋 锌褉懈褋胁邪懈胁邪薪懈械屑: `x //= 3`.
* `%=` - 芯褋褌邪褌芯泻 芯褌 写械谢械薪懈褟 褋 锌褉懈褋胁邪懈胁邪薪懈械屑: `x %= 3`.
* `**=` - 胁芯蟹胁械写械薪懈械 胁 褋褌械锌械薪褜 褋 锌褉懈褋胁邪懈胁邪薪懈械屑: `x **= 3`.

### <a name='锌褉懈褋胁邪懈胁邪薪懈械_锌械褉械屑械薪薪褘屑'></a> 袩褉懈褋胁邪懈胁邪薪懈械 锌械褉械屑械薪薪褘屑 [馃爼](#馃爼)

```python
room = 666                       # 袩褉懈褋胁邪懈胁邪薪懈械 褑械谢芯谐芯 褔懈褋谢邪

sum += 1                         # 袛芯斜邪胁谢械薪懈械 泻 写邪薪薪褘屑 锌械褉械屑械薪薪芯泄 褋 锌褉懈褋胁邪懈胁邪薪懈械屑
num_1 = num_2 = 3                # 袩褉懈褋胁邪懈胁邪薪懈械 写胁褍屑 锌械褉械屑械薪薪褘屑 芯写懈薪 芯斜褗械泻褌
swap1, swap2 = 8, 9              # 袦薪芯卸械褋褌胁械薪薪芯械 锌褉懈褋胁邪懈胁邪薪懈械
swap1, swap2 = swap2, swap1      # 袪芯泻懈褉芯胁泻邪 芯斜褗泻褌芯胁 屑械卸写褍 锌械褉械屑械薪薪褘屑懈
```

### <a name='褌懈锌褘_写邪薪薪褘褏'></a> 孝懈锌褘 写邪薪薪褘褏 [馃爼](#馃爼)

* `int` - 褑械谢褘械 褔懈褋谢邪: `some = 1`.
* `float` - 胁械褖械褋褌胁械薪薪褘械 褔懈褋谢邪: `some = 1.12`.

* `bool` - 斜褍谢械胁褘械: `some = True` 懈谢懈 `some = False`.

### <a name='懈写械薪褌懈褎懈泻邪褑懈褟_写邪薪薪褘褏'></a> type() [馃爼](#馃爼)

### <a name='锌褉械芯斜褉邪蟹芯胁邪薪懈械_写邪薪薪褘褏'></a> 袩褉械芯斜褉邪蟹芯胁邪薪懈械 写邪薪薪褘褏 [馃爼](#馃爼)

* `int()` - 锌褉械芯斜褉邪蟹芯胁褍械褌 胁 褑械谢褘械 褔懈褋谢邪.
* `float()` - 锌褉械芯斜褉邪蟹芯胁褍械褌 胁 胁械褖械褋胁械薪薪褘械 褔懈褋谢邪.
* `str()` - 锌褉械芯斜褉邪蟹芯胁褍械褌 胁 褋褌褉芯泻懈.
```python
num1 = 16                        # 袩械褉械屑械薪薪邪褟 褋 褑械谢褘屑 褔懈褋谢芯屑
num2 = 33.45                     # 袩械褉械屑械薪薪邪褟 褋 胁械褖械褋胁械薪薪褘屑 褔懈褋谢芯屑

i_num = int(num2)                # 袩褉械芯斜褉邪蟹芯胁邪薪懈械 胁械褖械褋褌胁械薪薪芯谐芯 褔懈褋谢邪 胁 褑械谢芯械
s_num = str(num1 + num2)         # 袩褉械芯斜褉邪蟹芯胁邪薪懈械 褔懈褋械谢 胁 褋褌褉芯泻褍
```

> 袙邪卸薪芯 蟹薪邪褌褜, 械褋谢懈 胁 褋褌褉芯泻械 `str` 懈屑械褞褌褜褋褟 斜褍泻胁械薪薪褘械 褋懈屑胁芯谢褘, 邪 薪械 褑懈褎褉褘, 褌芯 锌褉械芯斜褉邪蟹芯胁邪褌褜 胁 褑械谢芯褔懈褋谢械薪薪芯械 懈谢懈 胁械褖械褋褌胁械薪薪芯械 蟹薪邪褔械薪懈械 薪械 锌芯谢褍褔懈褌褜褋褟.

### <a name='print()'></a> print() [馃爼](#馃爼)

### <a name='锌械褉械薪芯褋_写谢懈薪薪芯泄_褋褌褉芯泻懈_泻芯写邪'></a> 袩械褉械薪芯褋 写谢懈薪薪芯泄 褋褌褉芯泻懈 泻芯写邪 [馃爼](#馃爼)

`\` - 褉邪蟹写械谢褟械褌 写谢懈薪薪褍褞 褋褌褉芯泻褍 泻芯写邪 薪邪 薪械褋泻芯谢褜泻芯:

```python
result = 3 + 5 * \
1 + 3 - 8

褌械泻褋褌'
print(text)
```

袙 泻褉褍谐谢褘褏 褋泻芯斜泻邪褏 褉邪蟹写械谢懈褌械谢褜 薪械 懈褋锌芯谢褜蟹褍械褌褋褟:

```python
mondey = 10.6 + 45.66 + 77
print('袩褉芯写邪卸懈 胁 锌芯薪械写械谢褜薪懈泻 褋芯褋褌邪胁懈谢懈',
mondey, '写芯谢谢邪褉芯胁')

num = (5 * 103.58 - 47.13 + 55.45 *
```

### <a name='褝泻褉邪薪懈褉芯胁邪薪薪褘械_褋懈屑胁芯谢褘'></a> 协泻褉邪薪懈褉芯胁邪薪薪褘械 褋懈屑胁芯谢褘 [馃爼](#馃爼)

* `\n` - 锌械褉械胁芯写懈褌 锌芯蟹懈褑懈褞 锌械褔邪褌懈 薪邪 褋谢械写褍褞褖褍褞 褋褌褉芯泻褍.
* `\t` - 锌械褉械胁芯写懈褌 锌芯蟹懈褑懈褞 锌械褔邪褌懈 薪邪 褋谢械写褍褞褖褍褞 谐芯褉懈蟹芯薪褌邪谢褜薪褍褞 锌芯蟹懈褑懈褞 褌邪斜褍谢褟褑懈懈.
* `\'` - 锌械褔邪褌邪械褌 芯写懈薪邪褉薪褍褞 泻邪胁褘褔泻褍.
* `\"` - 锌械褔邪褌邪械褌 写胁芯泄薪褍褞 泻邪胁褘褔泻褍.
* `\\` - 锌械褔邪褌邪械褌 芯斜褉邪褌薪褍褞 泻芯褋褍褞 褔械褉褌褍.

```python
'袩械褉胁邪褟 褋褌褉芯泻邪\n袙褌芯褉邪褟 褋褌褉芯泻邪'
'袦械卸写褍\t芯斜褗械泻褌邪屑懈\t褌邪斜褍谢褟褑懈褟'
'袙褘写械谢褟械屑 \'芯斜褗械泻褌\' 胁 泻邪胁褘褔泻懈'
```

### <a name='f-褋褌褉芯泻懈'></a> F-褋褌褉芯泻懈 [馃爼](#馃爼)

* `f'袩褉懈胁械褌 屑懈褉!'` - 芯斜褘褔薪芯械 褎芯褉屑邪褌懈褉芯胁邪薪懈械.
* `f'{some}'` - 胁褋褌邪胁泻邪 锌械褉械屑械薪薪芯泄.
* `f'{50 - 20 * 2}'` - 胁褋褌邪胁泻邪 胁褘褉邪卸械薪懈褟.
* `f'{53.3 * 55.7 :e}'` - 褎芯褉屑邪褌懈褉芯胁邪薪懈械 胁 薪邪褍褔薪芯泄 薪芯褌邪褑懈懈.
* `f'{10 - 7 :d}'` - 褎芯褉屑邪褌懈褉芯胁邪薪懈械 褑械谢芯谐芯 褔懈褋谢邪.
* `f'{17.11 - 4.11 :f}'` - 褎芯褉屑邪褌懈褉芯胁邪薪懈械 胁械褖械褋褌胁械薪薪芯谐芯 褔懈褋谢邪.
* `f'{18.13 / 2.14 :.2f}'` - 芯泻褉褍谐谢械薪懈械 胁械褖械褋褌胁械薪薪芯谐芯 褔懈褋谢邪 写芯 褋芯褌褘褏.
* `f'{45.76 * 22.12 :,}'` - 胁褋褌邪胁泻邪 蟹邪锌褟褌褘褏 胁 泻邪褔械褋褌胁械 褉邪蟹写械谢懈褌械谢褟.
* `f'{0.5 :%}'` - 褎芯褉屑邪褌懈褉芯胁邪薪懈械 胁械褖械褋褌胁械薪薪芯谐芯 褔懈褋谢邪 胁 锌褉芯褑械薪褌褘.
* `f'{some :10}'` - 褍泻邪蟹邪薪懈械 屑懈薪懈屑邪谢褜薪芯泄 褕懈褉懈薪褘 锌芯谢褟.
* `f'{some :<^>10}'` - 胁褘褉邪胁薪懈胁邪薪懈械 蟹薪邪褔械薪懈泄 胁 锌芯谢械.
* **[胁褘褉邪胁薪懈胁邪薪懈械] [褕懈褉懈薪邪] [,] [褌芯褔薪芯褋褌褜] [褌懈锌]** - 锌褉邪胁懈谢褜薪邪褟 锌芯褋谢械写芯胁邪褌械谢褜薪芯褋褌褜.

### <a name='芯锌械褉邪褌芯褉褘_褋褉邪胁薪械薪懈褟'></a> 袨锌械褉邪褌芯褉褘 褋褉邪胁薪械薪懈褟 [馃爼](#馃爼)

* `==` - 褉邪胁薪芯: `x == y`.
* `!=` - 薪械 褉邪胁薪芯: `x != y`.
* `>` - 斜芯谢褜褕械 褔械屑: `x > y`.
* `<` - 屑械薪褜褕械 褔械屑: `x < y`.
* `>=` - 斜芯谢褜褕械 褔械屑 懈谢懈 褉邪胁薪芯: `x >= y`.
* `<=` - 屑械薪褜褕械 褔械屑 懈谢懈 褉邪胁薪芯: `x <= y`.

### <a name='谢芯谐懈褔械褋泻懈械_芯锌械褉邪褌芯褉褘'></a> 袥芯谐懈褔械褋泻懈械 芯锌械褉邪褌芯褉褘 [馃爼](#馃爼)

* `and` - 胁芯蟹胁褉邪褖邪械褌 蟹薪邪褔械薪懈械 True 械褋谢懈 芯斜邪 褍褌胁械褉卸写械薪懈褟 胁械褉薪褘: `x > 5 and x < 10`.
* `or` - 胁芯蟹胁褉邪褖邪械褌 True 械褋谢懈 芯写薪芯 懈蟹 褍褌胁械褉卸写械薪懈泄 胁械褉薪芯: `x < 5 or x < 4`.
* `not` - 屑械薪褟械褌 褉械蟹褍谢褜褌邪褌, 胁芯蟹胁褉邪褖邪械褌 False 械褋谢懈 褉械蟹褍谢褜褌邪褌 True: `not(x < 5 and x < 10)`.

### <a name='芯锌械褉邪褌芯褉褘_锌褉懈薪邪写谢械卸薪芯褋褌懈'></a> 袨锌械褉邪褌芯褉褘 锌褉懈薪邪写谢械卸薪芯褋褌懈 [馃爼](#馃爼)

* `in` - 胁芯蟹胁褉邪褖邪械褌 True 械褋谢懈 锌芯褋谢械写芯胁邪褌械谢褜薪芯褋褌褜 锌褉懈褋褍褌褋褌胁褍械褌 胁 芯斜褗械泻褌械: `x in y`.
* `not in` - 胁芯蟹胁褉邪褖邪械褌 True 械褋谢懈 锌芯褋谢械写芯胁邪褌械谢褜薪芯褋褌褜 薪械 锌褉懈褋褍褌褋褌胁褍械褌 胁 芯斜褗械泻褌械: `x not in y`.

### <a name='褍褋谢芯胁薪褘械_芯锌械褉邪褌芯褉褘'></a> 校褋谢芯胁薪褘械 芯锌械褉邪褌芯褉褘 [馃爼](#馃爼)

* `if` - "械褋谢懈" 褍褋谢芯胁懈械 `True` 褌芯 斜褍写械褌 胁褘锌芯谢薪褟褌褋褟 胁谢芯卸械薪薪邪褟 懈薪褋褌褉褍泻褑懈褟:

```python
if 4 < 10:
	print('if 褋褉邪斜芯褌邪谢')
```

* `elif` - 写芯锌芯谢薪懈褌械谢褜薪芯械 "械褋谢懈", 泻芯褌芯褉芯械 褋褌邪胁懈褌褋褟 锌芯褋谢械 `if`, 懈 胁褘锌芯谢薪褟械褌褋褟 泻芯谐写邪 锌褉械写褘写褍褖懈械 褍褋谢芯胁懈褟 褟胁谢褟褞褌褋褟 `False`:

```python
x = 15
y = 9
if x == y:
	print('if 褋褉邪斜芯褌邪谢')
elif x > y:
	print('elif 褋褉邪斜芯褌邪谢')
```
* `else` - "懈薪邪褔械", 褋褌邪胁懈褌褋褟 锌芯褋谢械 `if` 懈 `elif`, 胁褘锌芯谢薪褟械褌褋褟 泻芯谐写邪 胁褋械 褍褋谢芯胁懈褟 `False`:

```python
a = 56
b = 30
if a < b:
	print('if 褋褉邪斜芯褌邪谢')
elif a == b:
	print('elif 褋褉邪斜芯褌邪谢')
else:
	print('else 褋褉邪斜芯褌邪谢')
```

### <a name='褑懈泻谢懈褔械褋泻懈械_芯锌械褉邪褌芯褉褘'></a> 笑懈泻谢懈褔械褋泻懈械 芯锌械褉邪褌芯褉褘 [馃爼](#馃爼)

* `while` - 褑懈泻谢 褋 褍褋谢芯胁懈械屑 锌芯胁褌芯褉械薪懈褟:

```python
x = 1
b = 11
while x < b:
	print(x)
	x += 1
print('袣芯薪械褑')
```

* `for` - 褑懈泻谢 褋芯 褋褔械褌褔懈泻芯屑 锌芯胁褌芯褉械薪懈褟:

```python
for num in [1, 2, 3, 4, 5]:
	print(num)
print('袣芯薪械褑')
```

小 锌芯屑芯褖褜褞 褎褍薪泻褑懈懈 `range()` 屑芯卸薪芯 蟹邪写邪褌褜 写懈邪锌邪蟹芯薪:

```python
for num in range(10, 0, -1):     # range(薪邪褔邪谢芯, 泻芯薪械褑, 褕邪谐)
	print(num)
print('袣芯薪械褑')
```

袠薪褋褌褉褍泻褑懈懈 褍锌褉邪胁谢械薪懈褟 褑懈泻谢芯屑:

* `break` - 锌褉械褉褘胁邪械褌 褑懈泻谢, 胁 泻芯褌芯褉芯屑 芯薪 芯斜褗褟胁谢械薪:

```python
x = 1
b = 101
while x <= b:
	if x == 51:
		break
	print(x)
	x += 1
print('袣芯薪械褑')
```

* `continue` - 锌褉芯锌褍褋泻邪械褌 芯褋褌邪胁褕械械褋褟 褌械谢芯 褑懈泻谢邪 褌械泻褍褖械泄 懈褌械褉邪褑懈懈:

```python
for val in '褋褌褉芯泻邪':
    if val == '芯':
        continue
    print(val)
print('袣芯薪械褑')
```

### <a name='len()'></a> len() [馃爼](#馃爼)

`len()` - 胁芯蟹胁褉邪褖邪械褌 写谢懈薪褍 (泻芯谢懈褔械褋褌胁芯 褝谢械屑械薪褌芯胁) 胁 芯斜褗械泻褌械:

```python
len('袩褉懈胁械褌 屑懈褉!')               # 袙褘胁芯写 斜褍写械褌 11
```

袗褉谐褍屑械薪褌 屑芯卸械褌 斜褘褌褜 锌芯褋谢械写芯胁邪褌械谢褜薪芯褋褌褜褞, 褌邪泻芯泄 泻邪泻 褋褌褉芯泻邪, 斜邪泄褌褘, 泻芯褉褌械卸, 褋锌懈褋芯泻 懈谢懈 写懈邪锌邪蟹芯薪 懈谢懈 泻芯谢谢械泻褑懈械泄.
