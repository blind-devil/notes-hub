* `-=` - –≤—ã—á–∏—Ç–∞–Ω–∏–µ —Å –ø—Ä–∏—Å–≤–∞–∏–≤–∞–Ω–∏–µ–º: `x -= 3`.
* `*=` - —É–º–Ω–æ–∂–µ–Ω–∏–µ —Å –ø—Ä–∏—Å–≤–∞–∏–≤–∞–Ω–∏–µ–º: `x *= 3`.
* `/=` - –¥–µ–ª–µ–Ω–∏–µ —Å –ø—Ä–∏—Å–≤–∞–∏–≤–∞–Ω–∏–µ–º: `x /= 3`.
* `//=` - —Ü–µ–ª–æ—á–∏—Å–ª–µ–Ω–Ω–æ–µ –¥–µ–ª–µ–Ω–∏–µ —Å –ø—Ä–∏—Å–≤–∞–∏–≤–∞–Ω–∏–µ–º: `x //= 3`.
* `%=` - –æ—Å—Ç–∞—Ç–æ–∫ –æ—Ç –¥–µ–ª–µ–Ω–∏—è —Å –ø—Ä–∏—Å–≤–∞–∏–≤–∞–Ω–∏–µ–º: `x %= 3`.
* `**=` - –≤–æ–∑–≤–µ–¥–µ–Ω–∏–µ –≤ —Å—Ç–µ–ø–µ–Ω—å —Å –ø—Ä–∏—Å–≤–∞–∏–≤–∞–Ω–∏–µ–º: `x **= 3`.

#### <a name='2.3'></a> –û–ø–µ—Ä–∞—Ç–æ—Ä—ã —Å—Ä–∞–≤–Ω–µ–Ω–∏—è [ü†ï](#2)

* `==` - —Ä–∞–≤–Ω–æ: `x == y`.
* `!=` - –Ω–µ —Ä–∞–≤–Ω–æ: `x != y`.
* `>` - –±–æ–ª—å—à–µ —á–µ–º: `x > y`.
* `<` - –º–µ–Ω—å—à–µ —á–µ–º: `x < y`.
* `>=` - –±–æ–ª—å—à–µ —á–µ–º –∏–ª–∏ —Ä–∞–≤–Ω–æ: `x >= y`.
* `<=` - –º–µ–Ω—å—à–µ —á–µ–º –∏–ª–∏ —Ä–∞–≤–Ω–æ: `x <= y`.

#### <a name='2.4'></a> –õ–æ–≥–∏—á–µ—Å–∫–∏–µ –æ–ø–µ—Ä–∞—Ç–æ—Ä—ã [ü†ï](#2)

* `not` - –º–µ–Ω—è–µ—Ç —Ä–µ–∑—É–ª—å—Ç–∞—Ç, –≤–æ–∑–≤—Ä–∞—â–∞–µ—Ç `False` –µ—Å–ª–∏ —Ä–µ–∑—É–ª—å—Ç–∞—Ç `True`: `not(x < 5 and x < 10)`.

#### <a name='2.5'></a> –û–ø–µ—Ä–∞—Ç–æ—Ä—ã –ø—Ä–∏–Ω–∞–¥–ª–µ–∂–Ω–æ—Å—Ç–∏ [ü†ï](#2)

* `in` - –≤–æ–∑–≤—Ä–∞—â–∞–µ—Ç `True` –µ—Å–ª–∏ –ø–æ—Å–ª–µ–¥–æ–≤–∞—Ç–µ–ª—å–Ω–æ—Å—Ç—å –ø—Ä–∏—Å—É—Ç—Å—Ç–≤—É–µ—Ç –≤ –æ–±—ä–µ–∫—Ç–µ: `x in y`.
